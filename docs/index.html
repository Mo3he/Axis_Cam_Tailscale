WireGuard Encryption

Built on Tailscale's WireGuard-based tunnel for fast, modern, and auditable encryption on every connection.

⚡

No Root Required

Runs in user-space networking mode on Axis OS 12+. No need to enable root access on your camera.

📦

Simple EAP Install

Upload the .eap file through your camera's web interface. Start the app and authenticate - done.

🔄

Weekly Auto-Updates

New EAP packages are automatically built and released every week when a new Tailscale version is available.

🌐

Headscale Compatible

The Custom variant supports self-hosted Headscale servers with configurable server URL and auth key.

🏗️

ARM & AARCH64

Supports both ARM (armv7hf) and AARCH64 architectures, covering a wide range of Axis camera models.

Installation

1
Download the EAP Grab the right .eap file for your camera architecture from the releases page above.
2
Upload to your camera Log into your Axis camera web interface and go to Apps → Add App. Upload the .eap file.
3
Start & authenticate Start the app, click Open to view logs and get your Tailscale authentication URL. Sign in and you're connected.
